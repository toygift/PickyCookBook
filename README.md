# 패스트캠퍼스 프로젝트 PickyCookbook

팀프로젝트 iOS개발자 3인이 의기투합하여 앱을 만들었습니다.


# 기능
- Facebook Login
- 레시피 만들기 (스텝별 과정 입력)
- 평가하기 기능
- 북마크 리스트 작성
